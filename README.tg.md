## Имкониятҳо

- Пайвасти мустақими **P2P** + рамзгузории **AES-256**
- **Ҳимоя аз қаллобон** (огоҳӣ, рӯйхати сиёҳи ID, шикоят)
- **Ройгон** — бе маҳдудияти рӯзона пас аз бақайдгирӣ
- Интерфейси бисёрзабона, ID-ҳои содда бо префикси **BRO**

## Фарқи мо аз дигарон

- **Ройгон, бе маҳдудкунӣ** — Пас аз бақайдгирии ройгон — бе маҳдудияти рӯзона ва бе равзанаҳои зӯроварона.
- **Сервери худӣ** — Трафик тавассути серверҳои мо меравад — таъхири камтар, маълумоти шумо ба ширкатҳои сеюм намеравад.
- **Ҳимоя аз қаллобон** — Анти-фроди дарунсохт, ки барномаҳои оддӣ надоранд.
- **Барои ҳама** — Интерфейси бисёрзабона ва ID-ҳои содда бо префикси BRO.
