## Возможности
- Прямое **P2P**-соединение — минимальная задержка
- Сквозное шифрование **AES-256**
- Встроенная **защита от мошенников**: предупреждения, чёрный список ID, жалобы
- **Бесплатно** после регистрации — без дневного лимита
- ID с префиксом **BRO**, мультиязычный интерфейс
